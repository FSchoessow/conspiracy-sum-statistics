# conspiracy-sum-statistics
Calculates the wonkey "conspiracy sum" to disprove the novelty of these.

# Usage
It's currently setup to the a timeframe of one year, if you want to see data for less/more than that simply go into the source code an change the <code>date</code> and <code>max_date</code> struct. If you chose to select an imperial time make sure that it's at least present in the <code>date</code>-struct.
